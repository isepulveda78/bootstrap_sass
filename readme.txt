Install npm and ruby

Run `npm init` in your terminal when you first create your project

Then run `npm install boostrap --save`

Create a main.scss file and a main.css file. Link the main.css in your index.html file.

Add @import "node_modules/bootstrap/scss/bootstrap"; to your main.scss and add all the varibles above this line.

To change the colors in a bootstrap theme, run sass main.scss main.css in the command line which will convert variables into the new
colors for your theme.
